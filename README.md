## вҡЎ What Is This?

> *In an era where a single viral post can shift public opinion within hours, the ability to instantly verify information is no longer a luxury вҖ” it's a necessity.*

**Fake News Detector** is an ML-powered classification engine that analyzes the linguistic fingerprint of any news article or social media post and determines its credibility with a probabilistic confidence score.

```
Input Text в”Җв”Җв–ә Preprocessing в”Җв”Җв–ә TF-IDF Vectorization в”Җв”Җв–ә Logistic Regression в”Җв”Җв–ә Verdict + Confidence
```

## рҹ§Ә How It Works

```
Step 1: INPUT
  в””в”Җ User pastes a news article or social media text

Step 2: PREPROCESSING
  в””в”Җ Lowercase вҶ’ strip HTML/URLs вҶ’ remove punctuation вҶ’ normalize whitespace

Step 3: VECTORIZATION
  в””в”Җ TF-IDF transforms text into a high-dimensional sparse feature vector
     Each feature = weighted importance of a word/phrase in the corpus

Step 4: CLASSIFICATION
  в””в”Җ Logistic Regression outputs P(Fake) and P(Real)
     Confidence < 65% вҶ’ flagged as "Uncertain"

Step 5: DISPLAY
  в””в”Җ Verdict card + probability bar chart rendered in Streamlit
```

## вҡ пёҸ Limitations

| Limitation | Details |
|:-----------|:--------|
| **Domain Bias** | Trained primarily on US political news; may underperform on tech, sports, or Indian news |
| **Context Blindness** | Cannot understand satire, sarcasm, or cultural irony |
| **Static Knowledge** | Model doesn't update with new events after training |
| **Short Texts** | Social media posts < 20 words yield lower confidence |
