# 📰 Fake News Detection System

An AI-powered web app that detects whether a news article or social media post is fake or real.

## 🚀 Features
- Detect fake vs real news
- Confidence score
- Handles uncertain predictions
- Clean and interactive UI

## 🧠 Model
- TF-IDF Vectorizer
- Logistic Regression

## ▶️ Run Locally

pip install -r requirements.txt  
python -m streamlit run app.py

## 🌐 Deployment
Use Streamlit Community Cloud for free deployment.
